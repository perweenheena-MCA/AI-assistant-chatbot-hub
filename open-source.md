# Open Source Contribution Guide

Thanks for contributing to **HackAI**! This project is beginner-friendly and designed to scale.

## How to contribute
1. Fork the repository.
2. Create a feature branch:
   - `git checkout -b blackboxai/your-feature`
3. Make changes.
4. Run tests / run the app locally (see README).
5. Commit with clear messages.
6. Push your branch.
7. Open a Pull Request.

## Suggested first contributions
- Improve UI/UX (animations, accessibility)
- Add more intents to `data/responses.json`
- Add new response templates
- Improve fuzzy matching thresholds or parsing

## Code style
- Keep code readable and beginner-friendly
- Add short comments for non-obvious logic
